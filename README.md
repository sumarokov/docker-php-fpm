# docker-php-fpm

Docker image of php-fpm 7.x built with most needed extensions

## Contents

Built on top of official `php:7.1-fpm`

### Extensions installed:

- gd
- intl
- pdo_mysql
- mbstring
- mcrypt
- zip
- bcmath
- soap
- apcu
- xdebug

all dev-packages required to build extensions are removed
